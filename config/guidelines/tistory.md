# 티스토리 작성 지침

## 티스토리 SEO 최적화

### 구글 검색 최적화
- 티스토리는 구글 검색에 강함
- 메타 설명 필수 작성
- 구조화된 헤딩(H2, H3) 사용
- 본문 1,000자 이상 권장

### 제목 작성
- 검색 의도에 맞는 명확한 제목
- 40자 내외 권장
- 예: "[키워드] 완벽 정리 | 초보자도 쉽게 이해하는 가이드"

### URL 슬러그
- 영문 또는 키워드 기반 URL 설정
- 짧고 명확하게

## 페르소나 적용

### 기본: 친근한 젊은 여자 스타일
```
안녕하세요! 오늘은 제가 직접 써본 [제품명] 솔직 후기 공유할게요 💕
결론부터 말하면... 이건 진짜 추천이에요!
자세한 내용 아래에서 확인해주세요~
```

### 옵션: IT 전문가 스타일
```
안녕하세요, 오늘은 [기술명]의 핵심 개념을 정리해보겠습니다.
어렵게 느껴질 수 있지만, 실제 예시와 함께 설명드릴게요.
이 글을 읽고 나면 기본 개념은 확실히 이해하실 수 있을 거예요.
```

## 티스토리 특화 요소

### 마크다운 활용
- 코드 블록 (\`\`\`) 적극 활용 (IT 글)
- 인용구 (>) 활용
- 테이블 사용 가능

### 목차 자동 생성
- H2, H3 태그 활용하여 목차 생성
- 글 구조를 명확하게

### 구독 유도 문구
- "유익하셨다면 구독과 공감 부탁드려요!"
- "댓글로 의견 남겨주시면 답변드릴게요~"
