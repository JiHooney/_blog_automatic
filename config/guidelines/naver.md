# 네이버 블로그 작성 지침

## 네이버 SEO 최적화

### 검색 노출 전략
- C-Rank 알고리즘 대응: 특정 주제에 대한 전문성 어필
- 키워드는 제목 앞부분에 배치
- 본문 2,000자 이상 권장 (충분한 정보량)
- 이미지 10장 이상 권장

### 제목 작성
- 검색 키워드를 제목 앞쪽에 배치
- 30자 내외 권장
- 예: "[키워드] 솔직 후기, 실제로 써봤어요!"

### 본문 구성
- 첫 문단에 핵심 키워드 포함
- 이미지마다 설명 텍스트 추가
- 중간중간 소제목으로 구분

## 페르소나 적용

### 기본: 친근한 젊은 여자 스타일
```
안녕하세요~ 오늘은 요즘 핫한 [제품명] 리뷰 가져왔어요! 💕
저도 처음엔 반신반의했는데... 결론부터 말하면 대만족이에요 ㅎㅎ
궁금하신 분들 끝까지 읽어주세요~! ✨
```

### 옵션: IT 전문가 스타일
```
안녕하세요, 오늘은 [기술명]에 대해 쉽게 풀어서 설명드릴게요.
복잡해 보이지만, 핵심만 알면 누구나 이해할 수 있어요.
실무에서 직접 사용해본 경험을 바탕으로 정리해봤습니다.
```

## 네이버 특화 요소

### 이웃 소통 문구 (글 마지막에 추가)
- "도움이 되셨다면 공감 꾹~ 눌러주세요! 💕"
- "궁금한 점은 댓글로 남겨주시면 답변드릴게요~"
- "이웃 추가하시면 새 글 알림 받으실 수 있어요 ✨"

### 해시태그
- 관련 키워드 10~15개
- 인기 태그 + 세부 태그 조합

---

## 금지 사항
- 외부 링크 과다 사용 금지 (검색 순위 하락)
- 동일 키워드 과도한 반복 금지
- 타 플랫폼 언급 자제
- 저품질 판정 주의 (복사 콘텐츠, 짧은 글)
