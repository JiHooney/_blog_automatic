# 워드프레스 작성 지침

## 워드프레스 SEO 최적화

### 글로벌 SEO 전략
- Yoast SEO / Rank Math 플러그인 최적화
- 스키마 마크업 활용
- 본문 2,000자 이상 권장
- 내부 링크 전략 수립

### 제목 작성
- 60자 이내 (구글 검색 결과 표시 기준)
- 파워 워드 활용 (Ultimate, Complete, Best 등)
- 예: "The Complete Guide to [키워드] | 2024 Updated"

### 메타 설명
- 155자 이내
- CTA 포함
- 핵심 키워드 자연스럽게 포함

### URL 슬러그
- 영문 소문자, 하이픈 구분
- 3~5단어 이내로 간결하게

## 페르소나 적용

### 기본: 친근한 젊은 여자 스타일
```
Hey everyone! 오늘은 제가 직접 사용해본 [제품명] 리뷰를 공유할게요 💕
솔직히 처음엔 기대 안 했는데... 결과는 대만족이었어요!
궁금하신 분들은 끝까지 읽어주세요~
```

### 옵션: IT 전문가 스타일
```
안녕하세요, 오늘은 [기술명]에 대해 심층적으로 다뤄보겠습니다.
이론적인 설명과 함께 실제 구현 예시도 준비했어요.
초보자도 이해할 수 있도록 단계별로 설명드릴게요.
```

## 워드프레스 특화 요소

### 구텐베르크 블록 활용
- 이미지 갤러리 블록
- 인용구 블록
- 버튼 블록 (CTA)
- 컬럼 블록 (비교 콘텐츠)

### Featured Image (대표 이미지)
- 1200x630px 권장 (소셜 공유 최적화)
- 눈에 띄는 썸네일 필수
- ALT 태그 작성

### 카테고리 & 태그
- 카테고리: 대분류 (1~2개)
- 태그: 세부 키워드 (5~10개)

---

## CTA (Call to Action)

### 글 마지막 추가 문구
- "이 글이 도움이 되셨다면 공유해주세요!"
- "궁금한 점은 댓글로 남겨주세요 💬"
- "더 많은 유용한 정보를 원하시면 뉴스레터 구독하세요!"

## 주의 사항
- 네이버/티스토리와 동일 콘텐츠 금지 (리라이팅 필수)
- 이미지 최적화 (용량 압축)
- 모바일 반응형 확인
- GDPR 준수 (해외 독자 대상 시)
